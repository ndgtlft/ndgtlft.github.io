1. 기본 방침

프롬프트 메모는 사용자가 입력한 프롬프트를 기기 안에 저장하고 필요할 때 복사해 사용할 수 있도록 하는 앱입니다. 이 앱은 사용자의 개인정보나 입력 내용을 개발자의 서버로 전송하지 않습니다.

2. 수집하는 정보

개발자는 이 앱을 통해 개인을 식별할 수 있는 정보, 입력된 프롬프트, 장르, 이용 상황, 기기 정보를 수집하지 않습니다.

3. 기기 안에 저장되는 정보

등록한 프롬프트, 제목, 장르, 업데이트 날짜, 복사 횟수, 즐겨찾기 상태, 표시 모드 설정은 앱 기능 제공을 위해 기기 안에 저장됩니다. 이러한 정보는 개발자에게 전송되지 않습니다.

4. 외부 서비스, 광고, 분석

이 앱은 광고, 접근 분석, 크래시 분석, 추적 SDK를 사용하지 않습니다. 계정 등록이나 외부 서버 동기화 기능도 제공하지 않습니다.

5. 문의 시 정보

사용자가 문의하는 경우 이메일 본문, 회신용 이메일 주소 등 사용자가 이메일로 보낸 정보를 확인합니다. 문의 대응 이외의 목적으로는 사용하지 않습니다.

6. Apple에 의한 정보 처리

App Store 다운로드, 구매, 크래시 리포트, 진단 정보 등은 Apple의 약관 및 개인정보 처리방침에 따라 처리될 수 있습니다.

7. 변경

앱 기능 추가나 법령 등의 변경에 따라 이 방침을 업데이트할 수 있습니다. 중요한 변경이 있는 경우 이 페이지에서 안내합니다.
